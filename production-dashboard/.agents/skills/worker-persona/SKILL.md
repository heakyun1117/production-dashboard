# worker-persona SKILL

## 목적
7개 공정 작업자 관점에서 UI/UX 이해도와 실사용성을 시뮬레이션한다.

## 대상 공정
1. 하판 프린팅
2. 상판 프린팅
3. 분주
4. 로우슬리팅
5. 50T 타발
6. 개별 슬리팅
7. 조립/검사

## 시뮬레이션 시나리오
- "내 공정 화면만 빠르게 찾을 수 있는가"
- "편차 방향을 X/Y 없이 이해할 수 있는가"
- "AI 코멘트가 실제 조치로 이어지는가"
- "탭 이동 후 데이터가 유지되는가"

## 체크리스트
- [ ] 내 공정 탭 진입이 직관적인가
- [ ] 숫자/차트가 27인치에서 충분히 읽히는가
- [ ] 보정 권고 문구가 이해 가능한 한국어인가
- [ ] 경고/NG 시 즉시 대응할 수 있는가

## 공정별 작업자 시뮬레이션 상세

### ① 하판 프린팅
- 핵심 질문: "지금 카본/절연/실버 위치가 기준 대비 얼마나 벗어났나?"
- 필요 정보: Row 1~12 좌우/상하 편차(mm), CHECK/NG 기준선, 보정 방향
- 조정 도구: Q값(회전), 좌우 오프셋, 상하 오프셋
- 특이사항: Q값은 Row별 위치에 따라 회전 효과가 다름. 가운데 Row는 영향 적고, 양 끝 Row에 영향 큼

### ② 상판 프린팅
- 핵심 질문: "트레이 1번/2번 각각 위치가 맞나?"
- 필요 정보: 하판과 동일 구조, 트레이별 분리 표시
- 특이사항: 하판 로직 재사용, 트레이 구분만 추가

### ③ 분주
- 핵심 질문: "시약 도포 위치와 양이 정상인가?"
- 필요 정보: 분주면적, 분주간격, 분주높이
- 특이사항: 면적/간격에 대한 CHECK/NG 기준은 별도 정의 필요

### ④ 로우슬리팅
- 핵심 질문: "슬리팅 위치가 Row별로 균일한가?"
- 필요 정보: 전체폭, 타발폭 Row별 편차
- 특이사항: ±0.1000mm 기준, 프린팅 보정 영향을 직접 받음

### ⑤ 50T 타발
- 핵심 질문: "타발홀 위치가 정확한가?"
- 필요 정보: 타발홀 좌우/상하 편차
- 특이사항: 타발홀이 조립 정렬 기준이므로 정밀도 중요

### ⑥ 개별 슬리팅
- 핵심 질문: "개별 스트립 분리 시 손상 없는가?"
- 필요 정보: 절단면 상태
- 특이사항: 확장 준비 단계

### ⑦ 조립/검사
- 핵심 질문: "하판-상판 정렬이 맞나? 양면테이프 위치 정상인가?"
- 필요 정보: 양면 좌우/상하 편차, L/C/R 3구역 마진
- 조정 도구: 상판 위치 조정, 양면테이프 위치 조정
- 특이사항: L(좌), C(중앙), R(우) 3구역으로 나눠 마진 확인. 한쪽만 벗어나면 기울기 문제

## 현장 작업자 만족도 평가 기준 (95점 이상 목표)

| # | 항목 | 기준 | 배점 | 측정 방법 |
|---|------|------|------|----------|
| 1 | 정보 접근성 | 화면 진입 후 3초 내 핵심 데이터(편차, 판정) 확인 가능 | 20점 | 요약카드+테이블이 스크롤 없이 보이는가 |
| 2 | 방향 직관성 | ←좌/우→, ↑상/하↓ 표기가 일관되고 혼동 없음 | 15점 | X/Y 용어가 UI 어디에도 없는가 |
| 3 | 보정 안내 | "어디를 얼마나 움직여라"가 한 문장으로 표현됨 | 20점 | correctionText()가 모든 편차에 적용되는가 |
| 4 | 시각화 품질 | 인라인 바에 CHECK/NG 기준선 표시, 치우침 도형 직관적 | 15점 | InlineDeviationBar, BiasCompass 존재 및 가독성 |
| 5 | 시뮬레이션 | 보정값 입력 → 즉시 Before/After 비교 가능 | 15점 | 슬라이더/입력 → 테이블 실시간 반영 |
| 6 | 긴급 대응 | NG 항목이 붉은 강조 + AI 코멘트에서 즉시 안내 | 15점 | NG행 빨간 배경 + AI코멘트 최상단 NG 안내 |

### 평가 절차
1. 각 태스크 완료 시 위 6항목을 자체 채점
2. Summary에 점수표 + 근거 포함
3. 95점 미만 시 미달 항목 자체 수정 후 재검증
4. 재검증 후에도 미달이면 사유 기재

## 확장 시뮬레이션 시나리오

### 시나리오 A: NG 발생 시 작업자 행동
1. 화면에서 빨간색 NG 확인
2. AI 코멘트에서 "Row X 좌우 +0.154mm NG, ←좌 0.154mm 보정 필요" 확인
3. 시뮬레이션 패널에서 보정값 입력
4. Before/After 비교로 보정 효과 확인
5. 설비에 보정값 적용

### 시나리오 B: CHECK 누적 시 작업자 행동
1. CHECK 3건 이상 → AI 코멘트에서 "추이 관찰 권장" 확인
2. 트렌드 확인 (같은 방향으로 계속 밀리는지)
3. 선제 보정 여부 판단

### 시나리오 C: 공정 간 보정 요청 수신
1. 관리자/다른 공정에서 "좌측 0.03mm 보정 필요" 알림 수신
2. 내 공정 마진 확인 (보정 여유 있는지)
3. 시뮬레이션으로 보정 후 내 공정 영향 확인
4. 수락/거절 판단

## 상판 프린팅 작업자

### 하판과의 차이
- 라인 1개 (A/B 구분 없음)
- 트레이 2개 (트레이1, 트레이2) — 각각 독립 보정
- 인쇄 2도 (카본→실버, 하판은 3도)

### 핵심 질문
- "내 트레이(1 또는 2) 편차 어때?"
- "트레이 간 편차 차이가 큰가?" (전체 보기에서 확인)
- "트레이별 보정값이 따로 계산되나?"

### 트레이별 보정 시나리오
- 트레이1만 NG → 트레이1 보정값만 수정, 트레이2는 유지
- 트레이1+2 모두 같은 방향 편차 → 공통 원인(판 자체) 의심
- 트레이 간 편차 반대 방향 → 트레이 개별 정렬 문제

## 수동조립기 작업자

### 공정 특성
- 설비 4대 (수동조립기 #1~#4)
- **사람이 직접 조립** → 설비 편차 + 작업자 편차 동시 존재
- 상판 + 양면(하판)을 결합하는 공정
- 측정 포인트: 좌우 타발홀 12개의 편차값

### 핵심 질문
- "내 설비(#1~#4)의 조립 편차 어때?"
- "같은 설비에서 작업자 A vs B 편차 차이가 큰가?"
- "설비 자체 문제인가, 내 조립 습관 문제인가?"

### 편차 분석 관점
- 설비별 평균 편차 비교 → 설비 자체 문제 식별
- 동일 설비 내 작업자별 편차 비교 → 작업자 습관 식별
- 타발홀 12개 중 특정 위치만 편차 → 정렬 지그 문제

### 보정 방법
- 설비 보정: 지그 위치 조정 (설비 담당자)
- 작업자 보정: 조립 습관 교정 (위치 맞추기, 눌림 힘 일관성)
- 타발홀 좌/우 12개 → 좌측 6개 평균 vs 우측 6개 평균 비교로 좌우 치우침 판단
