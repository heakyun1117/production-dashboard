# manager-persona SKILL

## 목적
관리자 관점에서 전체 공정 모니터링, 연동 판단, 보고서 공유 흐름을 검증한다.

## 핵심 관점
- 공정 간 마진 공유 상태 파악
- 프린팅↔로우슬리터↔조립 연동 시나리오 판단
- 트렌드 기반 선제 조치
- 보고서 생성/공유 완결성

## 시나리오
1. 조립 마진 부족 감지
2. 프린팅에 보정 요청 발행
3. 로우슬리터 영향 확인
4. Before/After 비교
5. PDF 보고서 생성 및 전달

## 체크리스트
- [ ] 공정별 상태를 한 화면에서 요약 확인 가능한가
- [ ] 보정 요청-수신-결과 추적이 가능한가
- [ ] 트렌드 경고를 선제조치에 활용 가능한가
- [ ] 보고서가 의사결정에 필요한 정보를 담는가

## 연동 시나리오 상세 (3종)

### 시나리오 ① 조립 마진 부족 → 프린팅 보정
1. 조립 탭에서 L/C/R 마진 중 하나가 CHECK 이하로 감소
2. 관리자 종합 탭에서 위험도 "높음" 표시
3. 프린팅에 보정 요청 발행 (방향 + 크기 명시)
4. 프린팅 작업자가 시뮬레이션으로 확인 후 적용
5. 적용 후 조립 마진 개선 여부 추적

### 시나리오 ② 편차 점진 악화 → 설비 점검
1. 트렌드 탭에서 SET 5개 연속 같은 방향 편차 증가 감지
2. AI 코멘트 레벨3 "SET 295부터 카본 좌우 우측 밀림 지속, 스텐실 점검 필요"
3. 관리자가 설비 점검 지시
4. 점검 후 Before/After 비교

### 시나리오 ③ SET 모드 전환 판단
1. SET(셋팅) → TEST(테스트) → PRD(생산) 전환 기준
2. SET에서 전 Row OK + 마진 30% 이상 → TEST 전환 가능
3. TEST에서 3연속 시트 전 Row OK → PRD 전환 가능
4. PRD 중 NG 발생 → SET으로 복귀 판단

## 관리자 대시보드 요구사항

### 전공정 요약 패널
- 7개 공정 × (OK/CHECK/NG 개수) 매트릭스
- 위험 공정 빨간 강조
- 가장 최근 측정 시각 표시

### 연동 위험도 표시
- 프린팅↔로우슬리터↔조립 3공정의 마진 상태를 한눈에
- 위험도: 안전(초록) / 주의(노랑) / 위험(빨강)

### 보정 추적 이력
- 보정 요청 → 수락/거절 → 적용 → 결과 추적
- 타임라인 형태로 시각화

### 시뮬레이션 이력
- 누가 언제 어떤 보정값을 시뮬레이션했는지 기록
- 실제 적용 여부 표시

## 상판 관리 포인트
- 트레이1 vs 트레이2 편차 패턴 비교
- 하판과 상판 간 편차 경향 일치 여부 (일치하면 공통 원인)
- 상판은 1라인이므로 A/B 비교 불필요

## 수동조립기 관리 포인트

### 설비 간 비교 (핵심!)
- 4대 설비 편차를 한눈에 비교 → 특정 설비만 문제인지 전체 문제인지 판단
- 설비별 평균 편차 트렌드 추적 → 지그 마모 조기 감지
- 3초 내 "어느 설비가 문제인가" 판별 가능해야 함

### 작업자 관리
- 작업자별 편차 추적 → 교육 필요 인원 식별
- 같은 설비에서 작업자 교대 전/후 편차 비교
- 특정 작업자 편차 과다 → 재교육 또는 배치 변경

### 연동 시나리오
- 수동조립 마진 부족 → 상판/하판 프린팅 보정 요청 검토
- 설비 #N 편차 점진 악화 → 지그 점검 알림
- 조립 편차가 최종 검사 불량률에 미치는 영향 추적


## 조립/검사 관리 포인트

### 최종 관문
- 조립/검사는 프린팅→로우슬리터→조립으로 이어지는 공정 마진의 **최종 확인 단계**
- 여기서 문제 발견 시 앞 공정(프린팅)으로 보정 요청

### Row별 잔여율 모니터링
- Row 1~12 중 잔여율 <20%인 Row 즉시 식별
- 잔여율 하락 추세 감지 → 선제 보정 판단
- 3초 내 "어느 Row가 위험한가" 판별 가능해야 함

### 보정 경로 판단 (관리자 핵심 역할)
| 상황 | 판단 | 보정 경로 |
|------|------|----------|
| Row 7만 CHECK | 국소 문제 | 조립기 Row 7 개별 보정 |
| 전체 Row 우측 치우침 | 전역 문제 | 프린팅 전역 보정 |
| Row 5~8 집중 + 전체 경향 | 복합 문제 | 프린팅 전역 + 조립기 추가 |
| 잔여율 20% 미만 Row 다수 | 긴급 | 즉시 보정 + 원인 추적 |

### 공정 간 연동
- 조립 마진 부족 감지 → 프린팅에 전역 보정 요청
- 프린팅 보정 후 → 조립 결과 Before/After 확인
- 잔여율 변화로 보정 효과 정량적 평가
