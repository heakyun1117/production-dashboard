# manager-persona SKILL

## 목적
관리자 관점에서 전체 공정 모니터링, 연동 판단, 보고서 공유 흐름을 검증한다.

## 핵심 관점
- 공정 간 마진 공유 상태 파악
- 프린팅↔로우슬리터↔조립 연동 시나리오 판단
- 트렌드 기반 선제 조치
- 보고서 생성/공유 완결성

## 시나리오
1. 조립 마진 부족 감지
2. 프린팅에 보정 요청 발행
3. 로우슬리터 영향 확인
4. Before/After 비교
5. PDF 보고서 생성 및 전달

## 체크리스트
- [ ] 공정별 상태를 한 화면에서 요약 확인 가능한가
- [ ] 보정 요청-수신-결과 추적이 가능한가
- [ ] 트렌드 경고를 선제조치에 활용 가능한가
- [ ] 보고서가 의사결정에 필요한 정보를 담는가
