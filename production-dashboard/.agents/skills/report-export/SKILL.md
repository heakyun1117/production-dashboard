# report-export SKILL

## 목적
jsPDF + html2canvas로 현장 보고서를 A4 가로 형식 PDF로 생성한다.

## 기술 규칙
- 캡처: html2canvas
- PDF: jsPDF
- 용지: A4 Landscape
- 한국어 폰트 임베딩 필수 (깨짐 방지)

## 보고서 구성
1. 표지
   - 제목, 일시, 라인/SET, 작성자
2. 본문
   - 공정별 핵심 지표
   - OK/CHECK/NG 요약
   - 공정 연동 코멘트
3. 부록
   - 원시 통계 테이블
   - 보정 이력

## 출력 품질
- 차트/표 폰트 최소 10pt 이상
- 색상 대비 확보
- 페이지 분할 시 표 제목 반복 권장
