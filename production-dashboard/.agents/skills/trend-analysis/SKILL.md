# trend-analysis SKILL

## 목적
SET 시계열 품질 변화를 추적하고 이상 신호를 조기 감지한다.

## 분석 항목
- 연속 CHECK 발생
- 연속 NG 발생
- 단조 증가/감소 추세
- 급격한 변동(스파이크)

## 통계 기법
1. 이동평균(MA)
2. 이동표준편차
3. 관리한계
   - 경고: ±2σ
   - 이상: ±3σ

## 이상감지 예시
- 최근 5 SET 중 CHECK 3회 이상 → 경고 코멘트
- NG 2회 연속 → 즉시 알림
- 편차가 한 방향으로 연속 증가 → 설비 점검 권고

## 출력 형식
- 라인차트 + 관리한계 밴드
- 요약 카드(추세/리스크/권장조치)
