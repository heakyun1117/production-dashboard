# chart-viz SKILL

## 목적
Recharts 단일 라이브러리로 공정 데이터 시각화를 표준화한다.

## 기본 규칙
- 차트 라이브러리는 **Recharts만 사용**
- 상태 색상 매핑:
  - OK `#2D68C4`
  - CHECK `#F59E0B`
  - NG `#EF4444`

## 차트 유형 가이드
1. 막대차트: 공정/포인트별 편차 비교
2. 산점도: 분포/이상치 확인
3. 라인차트: SET 추세 분석
4. 히트맵: Row/Column 편차 밀집 확인

## 부호 방향 반영
- 양수(+): 우측/위
- 음수(-): 좌측/아래
- 축 라벨, 범례, 툴팁에 X/Y 용어 금지

## UI 문구 예시
- `좌우 편차(←좌 / 우→)`
- `상하 편차(↑상 / 하↓)`

## 체크리스트
- 임계값선(±0.12, ±0.15) 표시 여부
- 막대 내부/옆 수치 라벨 표시 여부
- 모바일이 아닌 데스크톱 QHD 가독성 확인
